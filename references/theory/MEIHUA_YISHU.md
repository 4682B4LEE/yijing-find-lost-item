# 梅花易数理论

## 1. 梅花易数简介

梅花易数是中国古代占卜法之一，相传为宋代易学家邵雍所著。其特点是起卦方式灵活，可以根据时间、数字、声音、方位等多种方式起卦，被誉为"心法"。

在寻物应用中，梅花易数通过卦象推断失物的方位、环境和找回可能性。

## 2. 起卦方法

### 2.1 时间起卦法

使用丢失时间的年月日时数字进行计算：

- **上卦** = (年 + 月 + 日) % 8，若余数为 0 则取 8
- **下卦** = (年 + 月 + 日 + 时) % 8，若余数为 0 则取 8
- **动爻** = (年 + 月 + 日 + 时) % 6，若余数为 0 则取 6

**示例**：
2026年5月14日14时
- 上卦 = (2026 + 5 + 14) % 8 = 2045 % 8 = 5 → 巽卦
- 下卦 = (2026 + 5 + 14 + 14) % 8 = 2059 % 8 = 3 → 离卦
- 动爻 = (2026 + 5 + 14 + 14) % 6 = 2059 % 6 = 1 → 第1爻动

### 2.2 报数起卦法

使用用户随心报出的数字进行计算：

- **上卦** = 第一个数字 % 8，若余数为 0 则取 8
- **下卦** = 第二个数字 % 8，若余数为 0 则取 8
- **动爻** = (所有数字之和) % 6，若余数为 0 则取 6

**示例**：用户报数 7, 2
- 上卦 = 7 % 8 = 7 → 艮卦
- 下卦 = 2 % 8 = 2 → 兑卦
- 动爻 = (7 + 2) % 6 = 9 % 6 = 3 → 第3爻动

### 2.3 两种起卦方式的选择

- **优先使用报数起卦**：用户随心报出的数字更能反映其当下的"气"
- **时间起卦作为备选**：当用户未提供数字时使用
- **不可同时使用**：一次推演只使用一种起卦方式

## 3. 体用分析

### 3.1 体卦与用卦的定义

- **体卦**：代表失主（求测者）
- **用卦**：代表失物（所求之物）

### 3.2 确定体用

根据动爻位置确定体卦和用卦：

- **动爻 ≤ 3**（下卦动）：
  - 用卦 = 下卦（代表失物）
  - 体卦 = 上卦（代表失主）

- **动爻 > 3**（上卦动）：
  - 用卦 = 上卦（代表失物）
  - 体卦 = 下卦（代表失主）

**原理**：动爻所在的卦为"用"，因为动则变，变则代表所求之物的变化。

### 3.3 示例

上卦：艮（7），下卦：兑（2），动爻：3
- 动爻 ≤ 3，下卦动
- 用卦 = 下卦 = 兑（代表失物）
- 体卦 = 上卦 = 艮（代表失主）

## 4. 互卦计算

### 4.1 互卦的定义

互卦是从本卦中抽取爻位重新组合而成的卦，反映事物发展的中间过程。

### 4.2 计算方法

- **互卦上卦** = 本卦第 2、3、4 爻组成的卦
- **互卦下卦** = 本卦第 3、4、5 爻组成的卦

### 4.3 简化计算

在实际应用中，可使用简化方法：
- 互卦上卦 = ((上卦 + 下卦) × 2) % 8，若余数为 0 则取 8
- 互卦下卦 = ((上卦 + 下卦) × 3) % 8，若余数为 0 则取 8

## 5. 变卦计算

### 5.1 变卦的定义

变卦是将本卦的动爻阴阳翻转后得到的卦，反映事物的最终结果或变化趋势。

### 5.2 计算方法

- 将本卦中动爻所在的位置阴阳翻转（阳变阴，阴变阳）
- 上卦或下卦中动爻所在的卦发生变化

### 5.3 简化计算

- 若动爻 > 3（上卦动）：变卦上卦 = (上卦 + 动爻) % 8，下卦不变
- 若动爻 ≤ 3（下卦动）：变卦下卦 = (下卦 + 动爻) % 8，上卦不变

## 6. 卦象解读要点

### 6.1 本卦

反映当前状态，是推演的基础。

### 6.2 互卦

反映事物发展的中间过程，提示寻找过程中的线索。

### 6.3 变卦

反映最终结果或变化趋势，提示物品的最终状态。
