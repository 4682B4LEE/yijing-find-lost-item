# 五行生克关系

## 1. 五行基本概念

五行是中国古代哲学的重要组成部分，包括：金、木、水、火、土。五行之间存在相生和相克的关系，这种关系被广泛应用于易经推演中。

## 2. 五行相生关系

相生表示一种促进、滋养的关系：

```
金 → 水 → 木 → 火 → 土 → 金
```

| 生方 | 被生方 | 含义 |
|------|--------|------|
| 金 | 水 | 金生水，金属熔化为液体 |
| 水 | 木 | 水生木，水滋养植物生长 |
| 木 | 火 | 木生火，木材燃烧产生火 |
| 火 | 土 | 火生土，火焚烧后化为灰烬（土） |
| 土 | 金 | 土生金，土中蕴藏金属矿物 |

## 3. 五行相克关系

相克表示一种制约、抑制的关系：

```
金 → 木 → 土 → 水 → 火 → 金
```

| 克方 | 被克方 | 含义 |
|------|--------|------|
| 金 | 木 | 金克木，金属工具可以砍伐树木 |
| 木 | 土 | 木克土，植物根系穿透土壤 |
| 土 | 水 | 土克水，土可以阻挡水流 |
| 水 | 火 | 水克火，水可以灭火 |
| 火 | 金 | 火克金，火可以熔化金属 |

## 4. 寻物推演中的五行应用

### 4.1 体用五行关系判断

在寻物推演中，体卦代表失主，用卦代表失物。通过分析体卦五行与用卦五行的关系，可以判断寻回概率：

| 关系 | 条件 | 寻回概率 | 含义 |
|------|------|----------|------|
| 用生体 | 用卦五行生体卦五行 | 🟢 极高 | 失物主动来寻，很快就能找到 |
| 比和 | 用卦五行 = 体卦五行 | 🟢 较高 | 物品在常放处，仔细寻找即可 |
| 体克用 | 体卦五行克用卦五行 | 🟡 中等 | 费力可找到，按方位重点寻找 |
| 用克体 | 用卦五行克体卦五行 | 🟠 较低 | 寻找较困难，建议扩大范围 |
| 体生用 | 体卦五行生用卦五行 | 🔴 很低 | 物品可能已被移动或损坏 |

### 4.2 判断方法

1. **确定体卦五行**和**用卦五行**
2. 检查是否**比和**（五行相同）
3. 检查是否**用生体**（用卦五行生体卦五行）
4. 检查是否**体克用**（体卦五行克用卦五行）
5. 检查是否**用克体**（用卦五行克体卦五行）
6. 检查是否**体生用**（体卦五行生用卦五行）

### 4.3 示例

**示例 1**：
- 体卦 = 艮（土）
- 用卦 = 兑（金）
- 分析：土生金 → 体生用 → 寻回概率很低

**示例 2**：
- 体卦 = 震（木）
- 用卦 = 离（火）
- 分析：木生火 → 体生用 → 寻回概率很低

**示例 3**：
- 体卦 = 坎（水）
- 用卦 = 乾（金）
- 分析：金生水 → 用生体 → 寻回概率极高

## 5. 物品材质与五行对应

| 材质 | 五行 | 说明 |
|------|------|------|
| 金属 | 金 | 钥匙、硬币、珠宝、手表 |
| 木质/植物 | 木 | 木制品、纸张、布料、植物 |
| 水/液体 | 水 | 饮料、化妆品、玻璃制品 |
| 火/电子产品 | 火 | 手机、电脑、灯具、电器 |
| 土/陶瓷/矿物 | 土 | 陶瓷、石头、泥土、食物 |
| 布料/纸张 | 木 | 衣服、书籍、文件（归类为木） |
| 不确定 | 土 | 默认归类为土 |

## 6. 五行与方位对应

| 五行 | 对应方位 | 说明 |
|------|----------|------|
| 金 | 西北、正西 | 乾（西北）、兑（正西） |
| 木 | 正东、东南 | 震（正东）、巽（东南） |
| 水 | 正北 | 坎（正北） |
| 火 | 正南 | 离（正南） |
| 土 | 东北、西南 | 艮（东北）、坤（西南） |
