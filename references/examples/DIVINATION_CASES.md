# 寻物案例参考

## 案例 1：钥匙丢失（报数起卦）

### 用户输入

```
物品：家门钥匙
材质：金属
时间：2026-05-14 14:30
随心数字：7, 2
```

### 推演过程

1. **起卦**：
   - 上卦 = 7 % 8 = 7 → 艮卦
   - 下卦 = 2 % 8 = 2 → 兑卦
   - 动爻 = (7 + 2) % 6 = 3 → 第3爻动

2. **体用分析**：
   - 动爻 ≤ 3，下卦动
   - 用卦 = 下卦 = 兑（代表失物）
   - 体卦 = 上卦 = 艮（代表失主）

3. **方位推断**：
   - 用卦为兑，方位在正西
   - 环境特征：水边、缺口处、垃圾桶旁

4. **五行分析**：
   - 体卦艮（土） vs 用卦兑（金）
   - 土生金 → 体生用
   - 寻回概率：很低

### 推演结果

用卦：兑 (Dui)（代表失物，五行金）

🎯 寻回概率与建议
━━━━━━━━━━━━━━━━━━━━

寻回概率：很低

### 解读建议

"老夫已为你起得卦象。上卦为艮，下卦为兑，动爻在第三爻。

用卦为兑，兑为泽，方位在正西。此物多半在正西方向，靠近水边或有缺口之处。

互卦提示此物可能在废弃物堆附近，变卦则暗示可能在角落或柜子中。

五行关系为体生用，寻回概率很低。建议你检查正西方向的垃圾桶旁、洗手间、或厨房水槽附近。"

## 案例 2：猫咪走失（时间起卦）

### 用户输入

```
物品：猫咪
材质：布料/纸张（归类为木）
时间：2026-05-13 23:30
```

### 推演过程

1. **起卦**：
   - 上卦 = (2026 + 5 + 13) % 8 = 2044 % 8 = 4 → 震卦
   - 下卦 = (2026 + 5 + 13 + 23) % 8 = 2067 % 8 = 3 → 离卦
   - 动爻 = (2026 + 5 + 13 + 23) % 6 = 2067 % 6 = 3 → 第3爻动

2. **体用分析**：
   - 动爻 ≤ 3，下卦动
   - 用卦 = 下卦 = 离（代表失物）
   - 体卦 = 上卦 = 震（代表失主）

3. **方位推断**：
   - 用卦为离，方位在正南
   - 环境特征：明亮处、电子产品旁、火炉/厨房

4. **五行分析**：
   - 体卦震（木） vs 用卦离（火）
   - 木生火 → 体生用
   - 寻回概率：很低

### 推演结果

五行关系为体生用，寻回概率较低。建议你往正南方向的花园、草地寻找，同时留意附近是否有火炉、厨房或明亮的地方。"

## 案例 4：手机丢失（时间起卦 - 中等概率）

### 用户输入

```
物品：手机
材质：火/电子产品
时间：2026-05-14 10:00
```

### 推演过程

1. **起卦**：
   - 上卦 = (2026 + 5 + 14) % 8 = 2045 % 8 = 5 → 巽卦
   - 下卦 = (2026 + 5 + 14 + 10) % 8 = 2055 % 8 = 7 → 艮卦
   - 动爻 = (2026 + 5 + 14 + 10) % 6 = 2055 % 6 = 3 → 第3爻动

2. **体用分析**：
   - 动爻 ≤ 3，下卦动
   - 用卦 = 下卦 = 艮（代表失物）
   - 体卦 = 上卦 = 巽（代表失主）

3. **方位推断**：
   - 用卦为艮，方位在东北
   - 环境特征：角落、柜子/箱子里

4. **五行分析**：
   - 体卦巽（木） vs 用卦艮（土）
   - 木克土 → 体克用
   - 寻回概率：中等
